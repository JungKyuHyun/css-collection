# CANVAS-STUDY

이 세션은 <code>cavas</code> 공부의 결과물을 담습니다.

- 대부분의 <code>cavas</code> 예제 코드는 김종민님의 튜토리얼 강의에서 가져왔습니다.

- 예제 코드(JS)를 <code>React</code>로 컨버팅하면서, <code>React</code>에서 인터렉티브한 화면을 만들기 위한
  고민을 추가하였고,

- <code>Typescript</code> 적용을 통해 <code>cavas</code>의 인터페이스를 더 이해하기 위한 용도로 사용했습니다.
